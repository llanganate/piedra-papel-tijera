# Piedra, Papel o Tijera

## Descripción

Este proyecto corresponde a la asignatura de Lógica de Programación.

El software desarrollado consiste en un juego de Piedra, Papel o Tijera donde un jugador puede enfrentarse a la computadora (CPU).

## Funcionalidades

* Iniciar partida
* Seleccionar Piedra, Papel o Tijera
* Generación automática de jugada de la CPU
* Determinación del ganador
* Registro de resultados
* Historial de partidas

## Herramientas utilizadas

* Python
* Visual Studio Code
* Git
* GitHub
