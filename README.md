## Objetivo del sistema

Desarrollar un software funcional en Python que permita aplicar los conocimientos adquiridos durante la asignatura de Lógica de Programación, implementando estructuras condicionales, estructuras repetitivas, funciones y buenas prácticas de programación mediante un juego interactivo.

---

## Descripción del proyecto

El sistema consiste en un juego interactivo denominado **Piedra, Papel o Tijera**, en el que el usuario juega contra la computadora.

La computadora genera una opción aleatoria y el programa determina automáticamente el ganador de cada partida.

Además, el sistema permite registrar el historial de partidas y visualizar el marcador acumulado.
